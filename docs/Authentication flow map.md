# 11) Clean component structure

These auth pages can all share one wrapper:

```text
Auth Layout
├── Brand Header
├── Auth Card
│   ├── Title
│   ├── Description
│   ├── Form Fields
│   ├── Primary CTA
│   └── Secondary Links
└── Optional Side Panel / Illustration
```

---

# 12) Best visual direction

To match the UI you already made:

* keep the **dark navy brand sidebar tone**
* use **soft gray page background**
* use **rounded white auth cards**
* use the same **purple primary button**
* keep spacing roomy and minimal

That will make the auth flow feel like part of the same product.
